Creative Manager

This project is an idea generator for creatives who want to carefully manage their creative projects. It is important it doesn't feel like work. It consists of a flashcard randomizer and a canvas based mindmap feature.

  • 1. there is a database where keywords and description or other info are stored
  • 2. flashcard will take items from the database to display as keywords or cards to the user to choose (or randomly chosen automatically depending on the settings)
  • 3. there is a login page so that all the information for each user is private.
  • 4. after items are chosen it can be displayed in a Canvas area where the user can turn it into a mindmap and expand it further.
